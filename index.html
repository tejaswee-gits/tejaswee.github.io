🔧 SmartCV Engine

Every job application asks for a "tailored CV." But tailoring manually — reading the JD, rewriting bullets, repositioning skills — takes 30–45 minutes per role. Multiply that by 50+ applications and it becomes a full-time job in itself.

I didn't want to send generic CVs. But I also couldn't afford to spend hours rewriting the same document endlessly. So I built the solution.

What It Does:

  • Takes any Job Description as input
  • Cross-references it against a Master Career File (all experience, skills, metrics, stories)
  • Runs a smart compatibility check — does the background actually match this role?
  • Automatically writes a tailored Professional Summary, Experience Section, and Skills list — tuned to JD language
  • Outputs a clean, formatted PDF ready to send — in seconds, not hours

Multi-Node AI Pipeline:

Node Function
📥 Job Description Input Pastes raw JD text
🎯 Tailor Resume Strategy Analyzes JD vs. master career file, identifies fit and gaps
✍️ Write Resume Summary Generates a targeted professional summary
📝 Write Experience Rewrites work experience bullets in JD language
🧠 Extract Skills Surfaces the most relevant skills from full toolkit
📄 Tailored Resume Compiles everything into a polished, formatted CV

Supporting assets: Master Career Dossier, Standard Operating Procedure, Job Search & Visa Strategy document

Most people talk about "using AI for productivity." I built a system that actually does it — end-to-end, with real logic, not just a ChatGPT prompt. This project reflects how I think: if a process is repetitive, it should be automated. If data exists, it should inform the output. And if I'm going to do something 50 times, I'd rather engineer it once and run it forever.
