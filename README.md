## 🛠️ Tech Stack

This portfolio is intentionally built with **zero frameworks** — pure fundamentals for maximum control and performance:

```
HTML5          →  Semantic structure & SEO
Tailwind CSS   →  Utility-first responsive styling
Vanilla CSS    →  Custom animations, glassmorphism, gradients
JavaScript     →  Interactivity, chart logic, scroll handlers
Font Awesome   →  Icon library
Google Fonts   →  Playfair Display, Inter typography
Clearbit API   →  Company logos on positioning charts
```

**No React. No Next.js. No build step.**  
Just clean, performant code that loads instantly.
